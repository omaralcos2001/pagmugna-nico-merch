# Product images

Drop your product photos in this folder using these exact filenames. Until a file
exists, the store shows a green/marble placeholder for that product — so the page
looks good either way.

| Filename | Product |
|----------|---------|
| `cap-white.jpg`     | Trucker Cap — White/Green |
| `tee-obsidian.jpg`  | Performance Tee — Obsidian (dark teal/gold) |
| `tee-glacier.jpg`   | Performance Tee — Glacier (blue/white) |
| `tote-emerald.jpg`  | Event Tote Bag — Emerald (green banner) |
| `singlet-dark.jpg`  | Running Singlet — Obsidian (no collar) |
| `cap-green.jpg`     | Trucker Cap — Full Green |
| `singlet-light.jpg` | Running Singlet — Glacier |
| `arm-sleeves.jpg`   | Arm Sleeves |
| `buff.jpg`          | Neck Gaiter / Buff |
| `tote-sage.jpg`     | Heritage Tote Bag — Sage (green logo) |
| `payment-qr.png`    | **Your payment QR code** (GCash / Maya / bank) shown at checkout |
| `hero-map.jpg`      | **Route map** shown faded (green-tinted) behind the hero heading |
| `paubaya.mp3`       | Background song that plays from the hero button *(already added)* |

Tip: square images (1:1) look best. `.jpg` or `.png` both work — if you use PNG,
either rename to `.jpg` or update the path in `index.html`.
